# Job Portal

## Overview

Job Portal is a full-stack web application developed using React, Node.js, Express.js, and MongoDB. It provides a platform where students can explore job opportunities and recruiters can manage job postings.

## Features

- Home Page
- Login Page
- Register Page
- Jobs Page
- Recruiter Dashboard
- Student Dashboard
- About Page
- Responsive Navigation Bar
- Footer

## Technologies Used

### Frontend
- React.js
- React Router
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
